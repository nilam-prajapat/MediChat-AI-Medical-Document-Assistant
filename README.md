# MediChat-AI-Medical-Document-Assistant
MediChat AI is an intelligent medical document assistant built using Retrieval-Augmented Generation (RAG). It allows users to upload medical documents and interact with them through an AI-powered chat interface.

 Developed an AI-powered medical document assistant using **RAG (Retrieval-Augmented Generation)** architecture  
- Implemented **semantic search using FAISS vector database** and HuggingFace embeddings  
- Integrated **Groq LLM (LLaMA 3.1)** for context-aware response generation  
- Built an interactive UI using **Streamlit** with document upload, chat system, and history tracking  
- Designed a scalable pipeline for **document processing, chunking, and retrieval-based QA**  

---
Features--

Document upload (PDF, DOCX, TXT)
Semantic search (FAISS)
AI chat using Groq LLM
Chat history storage

🔹 Tech Stack--
Python
Streamlit
LangChain
FAISS
HuggingFace
Groq API

🔹 How to Run--
pip install -r requirements.txt
streamlit run app.py
